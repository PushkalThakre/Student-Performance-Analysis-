# Student-Performance-Analysis-
Student Performance Analysis is a data analytics project that examines student academic scores to identify subject-wise performance patterns, score distributions, and academic trends using Python data analysis and visualization techniques.
# Student Performance Analysis

## Overview

This project analyzes student academic performance using Python to identify score patterns, subject-wise trends, and overall performance insights.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Visualization of score patterns
5. Insight generation

## Key Insights

* Identification of strong and weak academic subjects
* Understanding score distribution among students
* Detection of performance trends
